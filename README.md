# Envelope Interaktif

## Assets bundler

### Requirement : 
- webpack ^5, please refer to : https://webpack.js.org/concepts/ for installation

### Setup 

```
npm install
```

or, if you are using yarn 

```
yarn 
```

### Build

```
npm run build
```
or, if you are using yarn 

```
yarn build
```

### Watch mode 
```
npm run watch
```
or, if you are using yarn 

```
yarn watch
```

### Development mode 
```
npm run dev
```
or, if you are using yarn 

```
yarn dev
```
